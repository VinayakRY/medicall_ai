# MediCall AI 🏥

An AI-powered Smart Healthcare Assistant built using FastAPI, Streamlit, Machine Learning, and AWS Cloud.

## Features

- AI Medical Chatbot
- Appointment Management
- Patient Database System
- Voice Assistant Support
- Streamlit Dashboard
- AWS EC2 Deployment
- Real-time API Integration

## Tech Stack

- Python
- FastAPI
- Streamlit
- SQLAlchemy
- MySQL
- AWS EC2
- Git & GitHub

## ⚡ Installation

```bash
git clone https://github.com/VinayakRY/medicall_ai.git

cd medicall_ai/backend

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

streamlit run dashboard.py
```
